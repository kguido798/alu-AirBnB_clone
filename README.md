🏠 AirBnB Clone – The Console
📌 Project Description

This project is the first step in building a full AirBnB clone web application.
It focuses on creating a command-line interpreter that manages AirBnB objects such as users and base models.

The console allows users to:

Create objects

Retrieve stored objects

Update object attributes

Delete objects

Persist data using JSON serialization

This foundation will later be extended with:

HTML/CSS templating

Database storage

RESTful API

Front-end integration
